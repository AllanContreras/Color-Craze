Color Craze - Frontend (Vite + React)

Requisitos:
- Node.js 18+ y npm

Instalar y ejecutar:

cd frontend
npm install
npm run dev

La app se lanzará en http://localhost:5173 por defecto y consumirá el backend en http://localhost:8080.

Notas:
- WebSocket: usa SockJS + STOMP en /color-craze/ws
- Variables y endpoints están hardcodeados para desarrollo local.
